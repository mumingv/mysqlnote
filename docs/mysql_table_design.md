# 数据表设计

## 设计注意点

- 数据表需要包含时间字段（create_time和update_time），用以查询和排查问题；
- 数据表需要包含删除标记字段（is_del），删除记录只需要设置标记即可，而不是真的删除记录；
- 每个字段都需要增加注释(Comment)；
- 枚举字段统一使用tinyint；
- bool字段统一使用tinyint；
- 所有字段都要是NOT NULL的，没有值的话需要自定义一个默认值；


### 给字段创建索引的几个条件

1. 该字段在where条件中经常使用；
2. 该字段的内容不是唯一的几个值（如：性别字段就不适合创建索引）；
3. 该字段内容不会频繁变化。
